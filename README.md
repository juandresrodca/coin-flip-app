# 🪙 Premium Coin Flip

A modern, interactive web application to help you make decisions with ease! This project focuses on beautiful UI/UX and smooth JavaScript animations.

## ✨ Features

- **3D Coin Animation**: A realistic flip that shows both Heads and Tails sides during rotation.
- **Decision Maker**: Type what you're deciding (e.g., "Pizza vs Burger") and let the coin choose for you.
- **Multi-Toss Sessions**: Set a number of tosses and trigger them manually one by one.
- **Premium Design**: 
  - Glassmorphism UI.
  - Animated particle background.
  - Responsive layout for mobile and desktop.
  - Vibrant gradients and modern typography.

## 🛠️ Purpose

This project was created to:
1. **Practice advanced JavaScript**: Handling 3D CSS transformations, canvas animations, and asynchronous logic.
2. **Easy Decisions**: Provide a simple, fun way to resolve choices quickly.

## 🚀 How to Run

Simply open `index.html` in any modern web browser.
